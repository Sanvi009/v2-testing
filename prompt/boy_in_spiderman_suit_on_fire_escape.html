How to Use This Prompt

Copy the exact prompt into your image model and include the provided reference image for the boy. For Midjourney, try `--ar 3:4 --v 5 --q 2` and lower stylize if it looks too artistic. For Stable Diffusion/SDXL, use 30–60 sampling steps, a strong guidance scale, face-restoration/postprocessing, and supply the negative prompt. Use a telephoto focal-length token (85mm or 135mm) and set high resolution (if available). Always check model rules about likeness and get consent when using real-person reference images.
